# Shop Computer Open Store

A computer shop and open store management system.

## Features

- Inventory management for computers and accessories
- Sales tracking and point of sale
- Customer management
- Supplier management
- Reporting and analytics

## Getting Started

1. Clone the repository
2. Install dependencies
3. Configure your environment
4. Run the application
